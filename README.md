# Responsive Tailwind Portfolio

A clean, responsive single-page developer portfolio built using Tailwind CSS v4.

## Features

- **Fluid Responsiveness**: Scales smoothly from mobile viewports up to large 1920px+ desktop monitors.
- **Sticky Section Timelines**: Interactive timeline grids for `2023` and `2024` with custom brand-specific pastel gradients for tech stack cards.
- **Sleek Client Testimonials**: Dynamic flex layout that automatically adjusts to screen width.

## Tech Stack

- HTML5
- Tailwind CSS v4 (CDN)

## Getting Started

Simply clone the repository and open `index.html` in your web browser, or run a local server:

```bash
# Clone the repo
git clone git@github.com:Suryacodeshere/Responsive-Tailwind-Portfolio-Practice.git

# Serve it locally
npx serve .
```
